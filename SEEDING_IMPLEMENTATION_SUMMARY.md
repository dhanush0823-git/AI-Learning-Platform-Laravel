## 📞 Support & Customization

### To Add More Courses:
1. Edit respective seeder file
2. Add course to `$additionalCourses` array
3. Run seeder: `php artisan db:seed --class=SeederName`

### To Modify Existing Courses:
1. Edit course in database through admin panel
2. Update video links as needed
3. Modify lesson content
4. Update questions

### To Create New Department:
1. Create new seeder file
2. Follow existing pattern
3. Add to DatabaseSeeder
4. Run migration and seeding
